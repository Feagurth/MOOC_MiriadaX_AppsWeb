Realiza un esquema sencillo de este documento HTML a su estructura tipo árbol, que resuma brevemente las relaciones directas entre elementos de la página, clarificando la jerarquía del árbol.

Dado el siguiente código para la creación de una página web...

Se puede definir la siguiente jerarquía en forma de arbol.

Tal como puede apreciarse en el diseño de la jerarquía, todos los elementos están contenidos dentro de la etiqueta HTML y son por tanto hijos suyos.

  • Los hijos de la etiqeuta HTML son:
    • La etiqueta HEAD, la cual tiene los siguientes hijos:
      • La etiqueta META
      • La etiqueta TITLE
    • La etiqueta BODY, la cual tiene los siguientes hijos:
      • La etiqueta HEADER, que tiene un único hijo:
        • La etiqueta IMG
      • La etiqueta NAV, que solo tiene como hijo una lista:
        • La etiqueta UL, que contiene 3 hijos
          • Etiqueta LI que hace referencia a el primer elemento de la lista
          • Etiqueta LI que hace referencia a el segundo elemento de la lista
          • Etiqueta LI que hace referencia a el tercer elemento de la lista
      • La etiqueta ARTICLE, la cual tiene los siguientes hijos:
        • La etiqueta H1 que es la cabecera del artículo
        • La etiqueta P que contiene el texto del artículo
      • La etiqueta FOOTER que no tiene hijos
